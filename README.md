# DiabetesAI--Prediction-System
Developed a Diabetes Prediction System using Python, Flask, Scikit-Learn, and Ensemble Learning. The application analyzes patient biomarkers and engineered health features to predict diabetes risk with probability scores, providing real-time clinical decision support through an interactive web dashboard.
